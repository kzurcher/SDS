# Sullivan's Diesel Solutions Website

Simple static website for Sullivan's Diesel Solutions using the provided shop logos and brand colors.

## Files

- `index.html` - main landing page
- `styles.css` - custom styling
- `script.js` - small footer script
- `logo-outline.jpeg` - clean line-art logo
- `logo-full.jpeg` - full-color truck logo

## Local preview

Open `index.html` in a browser.

## GitHub push

After creating a GitHub repo, run:

```powershell
git init
git add .
git commit -m "Initial Sullivan's Diesel Solutions website"
git branch -M main
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```
